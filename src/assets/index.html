Italian Politicians' Stats On Twitter


This is a project for "Web Programming, Design & Usability" - a university subject. The course aims to teach us how to use front-end and back-end languages and technologies. In this project, I have implemented a Twitter API system to analyze the use of Twitter by Italian Politicians of all coalitions. I fetch every day at midnight all metadata about profiles and tweets.
